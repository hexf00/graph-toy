# 数据结构定义

| field                    | struct  | description                              |
| ------------------------ | ------- | ---------------------------------------- |
| item.label               | String  | G6概念，文本                             |
| item.description         | String  | 描述文本                                 |
| edge.visible             | Boolean | 值false时不显示连线，其余时候显示连线    |
| node.labelCfg.style.text | String  | G6概念，显示的文本，优先级高于item.label |
